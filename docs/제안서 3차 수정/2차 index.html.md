## 법률구조서비스 플랫폼 개선 제안서 - 내용 정리

이 웹페이지는 **자치법연구원**과 **주식회사 포이**가 공동으로 제안한 법률구조서비스 플랫폼 개선 제안서입니다. 주요 내용을 체계적으로 정리하면 다음과 같습니다:

### 🎯 **프로젝트 개요**
- **목표**: One-Stop 맞춤형 법률구조 서비스 구현을 위한 AI 고도화 전략
- **정책 근거**: 국정과제 64-5 "법률구조서비스플랫폼 구축 및 사회적 취약계층 우선지원"
- **제안 기관**: 자치법연구원 × 주식회사 포이

### 🚨 **현재 문제점 (기존 RAG 시스템의 한계)**

#### 1. **의미적 연관성 부족**
- 키워드 매칭 중심의 단순 검색
- 법령-판례-절차 간의 논리적 연관관계 파악 불가
- 관련 없는 정보 제공, 맥락 무시

#### 2. **구조화되지 않은 정보 처리**
- 비정형 텍스트 덩어리로만 학습
- 법률 개념 간의 계층구조와 상호관계 무시
- 추상적이고 일반적인 답변만 가능

#### 3. **환각(Hallucination) 문제 심각**
- **법률 AI 도구 환각 발생률 33% 이상** (스탠포드 대학 연구)
- 일반 분야 대비 3배 높은 환각 발생률
- 잘못된 법률 정보 제공으로 인한 심각한 결과 초래

### 💡 **솔루션: Knowledge Graph 기반 데이터 구조화**

#### **핵심 접근법**
- **기존 데이터 구조화 우선**: 신규 데이터 수집보다 기존 데이터의 구조화를 통한 효율적 개선
- **Knowledge Graph RAG**: 단순 텍스트 유사도가 아닌 의미 기반 관계 탐색
- **능동형 대화 Agent**: 문제 해결 중심의 지능형 대화 시스템

#### **기존 RAG vs Knowledge Graph RAG 비교**
| 구분 | 기존 RAG | Knowledge Graph RAG |
|------|----------|-------------------|
| 검색 방식 | 텍스트 유사도 기반 | 의미 기반 관계 탐색 |
| 정보 연결 | 단편적 문서 검색 | 다중 홉 관계 추론 |
| 정확도 | 환각 발생 33% | 구조화로 환각 대폭 감소 |
| 확장성 | 선형적 개선 | 네트워크 효과로 기하급수적 향상 |
| 법률 추론 | 불가능 | 논리적 추론 경로 제공 |



#### **4계층 구조**
1. **🌐 사용자 접점 계층**: 웹 플랫폼, 모바일 앱
2. **🤖 능동형 대화 Agent 계층**: 의도 분석, 정보 수집, 추론 엔진, 답변 생성
3. **⚙️ 핵심 AI 엔진**: 한국어 특화 LLM, Knowledge Graph RAG, Vector DB
4. **📊 법률 지식 네트워크**: 법령 온톨로지, 판례 관계망, 지식 그래프
